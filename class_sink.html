Detailed Description

A sink is an object that writes the log message to the target.

Author
Christian Rapp

The virtual class Sink has to be implemented by each possible target. To add a new Sink to ealogger you have to provide an implementation of Sink::write_message

enabledWhether or not this sink is enabled
min_lvlThe minimum log severity

A Log message can have conversion patterns. Every Sink uses its own message pattern. Have a look at ConversionPattern for more details.

Internally std::strftime is used from header ctime to create formatted time strings. So for parameter Sink::datetime_pattern you have to use format specifiers that are recognised by strftime. See en.cppreference.com for details. For example "%H:%M:%S" returns a 24-hour based time string like 20:12:01
